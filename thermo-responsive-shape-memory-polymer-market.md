<p><strong>The Thermo-responsive Shape Memory Polymer Market Analysis by types is segmented into:</strong></p>
<p><ul><li>Type I</li><li>Type II</li></ul></p>
<p><p>The thermo-responsive shape memory polymer market can be categorized into two types: Type I and Type II. Type I polymers exhibit a single shape memory effect, returning to a pre-defined shape upon heating. These materials are often used in applications like self-healing materials and actuators. Type II polymers possess a dual-shape memory effect, allowing them to switch between two distinct shapes with different thermal triggers. This versatility makes Type II polymers suitable for advanced applications in robotics, biomedical devices, and smart textiles.</p></p>
